Click each maneuver to see how it affects each murmur. Understanding the mechanism is key.

Valsalva (Strain Phase)

↓ Preload → ↓ LV volume → ↓ venous return
HCM↑ LOUDER
MVP clickEarlier (murmur longer)
Aortic Stenosis↓ Softer
Mitral Regurgitation↓ Softer
VSD↓ Softer
HCM: ↓ LV volume → more LVOTO. MVP: ↓ LV → prolapse occurs sooner.

Standing Up

↓ Preload → ↓ venous return (blood pools in legs)
HCM↑ LOUDER
MVP clickEarlier
Aortic Stenosis↓ Softer
Mitral Regurgitation↓ Softer
Same mechanism as Valsalva — ↓ preload.

Squatting

↑ Preload + ↑ Afterload → ↑ LV volume
HCM↓ Softer (less obstruction)
MVP clickLater (murmur shorter)
Aortic Stenosis↑ Louder
Mitral Regurgitation↑ Louder
VSD↑ Louder
↑ LV volume → less LVOTO in HCM. ↑ SVR → ↑ MR gradient. MVP: fuller LV → prolapse delayed.

Handgrip

↑ Afterload (↑ SVR) → ↑ LV afterload
Mitral Regurgitation↑ Louder (↑ L→A gradient)
VSD↑ Louder
Aortic Regurgitation↑ Louder
HCM↓ Softer
Aortic Stenosis↓ Softer (less gradient across valve)
↑ SVR → regurgitant murmurs get louder; obstructive murmurs get softer.

Inspiration

↑ Venous return to RIGHT heart
Tricuspid Regurgitation↑ LOUDER (Carvallo sign)
Pulmonary Stenosis↑ Louder
Tricuspid Stenosis↑ Louder
Left-sided murmursNo significant change
Inspiration → ↑ right heart filling → all right-sided murmurs louder. Key for TR (Carvallo sign).

Amyl Nitrite (Vasodilator)

↓ Afterload + ↓ Preload (venous vasodilation)
HCM↑ Louder (↓ preload + afterload)
Aortic Stenosis↑ Louder (more gradient)
Aortic Regurgitation↓ Softer (↓ diastolic gradient)
Mitral Regurgitation↓ Softer (↓ L→A gradient)
Vasodilation → forward murmurs louder; regurgitant murmurs softer.
